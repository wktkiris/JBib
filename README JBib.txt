README for JBib.csl

日本語で書かれた文献リストを出力するための設定集です。辰巳智行さん（https://researchmap.jp/n8gozyt-c5s-2_4/）による「社会学評論スタイルガイド 第2版（非公式：邦文文献サブセット）」を改変。

#出力規則
・著者、編者等が複数名の場合は「、」で区切る。
・末尾はすべて「。」
・ページ数が複数にわたる場合省略せずに表記（「1113-1115頁」）
・初出情報はメモ欄に「初出: yyyy年」の形で記載すること。
・同著者の同年の文献は出版年の後にアルファベットをつけて表す。

・和書
　著者名（yyyy）『書名』、編集者名編『収録元』、出版社名、ページ（初出: {要編集}）。
・翻訳書
　Author Name（yyyy）『翻訳書名』訳者名、出版社名。
・論文
　著者名（yyyy）「題名」、『掲載誌』号数、巻数、ページ数頁。
・翻訳論文
　訳者名（yyyy）「翻訳論文の題名」、編者名編『掲載誌』、出版社名、翻訳論文の初ページ-終ページ
・本のチャプター
　著者名（yyyy）「題名」、編集者名編『書名』、出版社、ページ数頁。


上記以外の種類の文献の表記は辰巳さんの原本によります。
